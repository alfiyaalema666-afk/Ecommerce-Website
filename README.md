# 🌟 Lumora – Ecommerce Website

## 📖 Overview

Lumora is a modern and responsive ecommerce website developed using Django. It allows users to browse products, manage their shopping cart and wishlist, and securely authenticate using Django's built-in authentication system.

---

## ✨ Features

- User Registration & Login
- Product Categories
- Product Search
- Shopping Cart
- Wishlist
- User Profile
- Responsive Design
- Django Authentication
- Admin Panel
- Modern UI

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Lumora-Ecommerce-Website.git
```

Move into the project:

```bash
cd Lumora-Ecommerce-Website
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate it (Windows):

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Apply migrations:

```bash
python manage.py migrate
```

Run the server:

```bash
python manage.py runserver
```

Open your browser:

```
http://127.0.0.1:8000/
```
